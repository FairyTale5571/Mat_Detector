# Mat_Detector
## Пакет для ловли матерных слов в тексте

## Usage
```
go get github.com/FairyTale5571/Mat_Detector
```

```
if swears.ContainsMat(string) {
  fmt.Printf("Обнаружено матерное слово\n")
}
```
